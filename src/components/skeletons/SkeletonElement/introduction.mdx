# SkeletonElement Component

The `SkeletonElement` component provides a skeleton loader to represent a placeholder while content is loading. This component is useful for creating loading states for various UI elements such as text, charts, headers, and more.

## Usage

To use the `SkeletonElement` in your application, simply import it and pass a `$type` prop to define the style of the skeleton. The available types are:

- `text`
- `text-chart`
- `header`
- `title`
- `charts-body`
- `watchlist-title`
- `watchlist-header`
