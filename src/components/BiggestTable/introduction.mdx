# BiggestTable Component

The `BiggestTable` component displays either the biggest gainers or losers from the provided stock data. It supports loading states, dynamic content, and customizable configurations.

---

## Features

- Displays a table of stocks with key details (e.g., symbol, percentage change, short name).
- Supports two modes: `gainers` and `losers`.
- Loading skeleton for when data is being fetched.
- Customizable via props to control its behavior.
